# MIP-Project
The Smart Prayer Scheduler uses a microcontroller (ATmega32) to compute prayer times based on user input and display them on an LCD. A keypad is used to input the prayer times for the day (or days) that are used for time calculation. The system incorporates a lookup table to fetch the appropriate prayer times and then schedules the alarms accordingly or simply it alerts the user.The project also includes real-time updates and improvements for different prayer time algorithms.

The main file (.asm) is supposed to run using Atmel Studio, and then upload the hex file on ATmega32 microcontroller.
